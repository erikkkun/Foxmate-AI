FoxMate AI 使用说明
==================

欢迎使用 FoxMate AI！

系统要求：
- Windows 10/11 (64位)
- 至少2GB可用磁盘空间

快速开始：
1. 解压此ZIP文件到任意文件夹
2. 双击 "FoxMate AI.exe" 启动应用
3. 应用启动后，您将看到主页面
4. 点击 "Fox it!" 按钮可以启动专注监控功能

功能说明：
- 主页面：查看应用概览和快速启动
- My Account：查看和管理账户信息
- Weekly Report：查看每周专注度报告
- Your Fox：查看您的狐狸宠物状态
- Settings：应用设置

注意事项：
- 首次运行可能需要几秒钟加载
- 如果Windows显示安全警告，请选择"仍要运行"
- 建议将应用放在固定位置，不要移动文件夹

技术支持：
如有问题，请访问项目页面或联系开发者。
